Decision Guide

我该选哪个方案？

⚡

时效是第一优先

飞机在地面停场等待零件（AOG）、展会明天开幕、药品必须今晚到位……

这些情况下，每一分钟的延误都意味着巨大损失。费用已经不是最重要的考量。

→ 推荐 OBC 手提 / NFO 紧急空运
💰

成本控制是关键

货物不紧急，有2–7天的运输窗口，希望在合理时效内控制物流成本。

经济空运在速度与成本之间取得平衡，是日常货物的理想选择。

→ 推荐 经济空运 / 海运（大批量）
🔬

货物有特殊要求

危险品（DG）、温控医药品（冷链）、航空零件（AOG）、超大超重件……

普通渠道无法接受，需要专业资质与定制方案。

→ 推荐 NFO / SCP 专项服务
📦

小件高价值，容不得任何风险

样品、原型机、珠宝、重要合同文件……体积小但价值极高，绝对不能丢失或损坏。

OBC 专人全程随身携带，是唯一做到"零风险"的方式。

→ 推荐 OBC 手提服务
