## 要做什么

制作将会议记录文本输入后，由 Claude API 自动生成摘要和行动项，并保存到 Supabase 的工具。

## 学习内容

- **首次体验 Claude API**
- 系统提示词（System Prompt）设计
- 结构化输出（JSON 格式摘要 + 行动项）
- 流式传输（Streaming）显示生成过程
- 数据库持久化（保存到 Supabase）

## 完成标准

- [ ] 输入会议记录后 Claude 生成摘要
- [ ] 行动项以列表形式提取
- [ ] 生成过程以流式显示（逐字出现）
- [ ] 历史记录保存到 Supabase
- [ ] Vercel 部署成功
