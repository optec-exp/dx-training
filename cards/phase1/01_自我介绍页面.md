# 作品01 — 自我介绍页面

| | |
|---|---|
| **编号** | 01 |
| **工时** | S（入门）1〜2小时 |
| **Day** | Day 1 |
| **对象** | 全员 |

## 要做什么

用 HTML 制作一个展示自己姓名、职位、兴趣爱好的个人介绍页面，并发布到 Vercel。

## 学习内容

- HTML 基本结构（`h1` `p` `img`）
- 第一次向 Claude Code 下达指令
- `git add → commit → push` 操作
- Vercel 部署体验（获得公开 URL）

## 完成标准

- [ ] 页面显示姓名、职位、至少一张图片
- [ ] Vercel 部署成功，可通过 URL 访问

## 提交方式

```
git add .
git commit -m "作品01: 自我介绍页面"
git push
```

Vercel 自动部署后，将公开 URL 发送到 **#optec-ai-academy**。
