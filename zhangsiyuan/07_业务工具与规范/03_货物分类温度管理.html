货物分类与温度管理要求一览

航空货运特种货物分类 · 危险品 IATA 九大类 · 全链路温控管理规范——操作人员必备参考手册

🔍
📦
GEN

普通货物

▾

定义

  • 不需要特殊处理的标准货物，无危险、无温控、无特殊资质要求
  • 占航空货运总量约 70%，是最基础的货物类型

常见货物举例

  • 服装、鞋类、箱包、玩具、家居用品
  • 电子配件（非锂电驱动）、零部件、模具
  • 印刷品、样品、贸易展览品
  • 一般工业机械零件（非危险化学品）

操作要求

  • 按标准航空货运 SOP 操作，无额外特殊许可
  • 申报内容须与实货相符，避免查验
  • 重量/体积超过单箱 150kg 或单件超大尺寸须提前申报
IATA 编码 GEN 在系统录入时作为默认类型，若有任何特殊属性（含锂电、液体等）必须切换相应特种货物代码，不得保留 GEN。
☢️
DGR

危险品

▾

定义

  • 符合 IATA 危险品规则（DGR）定义的物品或物质，具有危险特性
  • 必须严格按照 IATA 第 65 版（2024 年版）DGR 手册操作
  • 九大类详见"危险品分类"标签页

前置条件（缺一不可）

  • 托运人须提供正确填写的《危险品申报单》（DG Declaration / Shipper's Declaration）
  • 货物须按 IATA 规定进行合规包装，贴附正确标签和标记
  • 货代须持有 IATA DGR 培训证书（至少 2 年一次更新）
  • 航空公司须事先接受（Prior Acceptance，PA），部分危险品不同航线接受条件不同

常见问题

  • UN 编号选择错误（如将锂离子电池混淆为锂金属电池）
  • 包装规格不符合 PI（Packing Instructions）要求
  • 申报数量超过限额（Q值）
  • 未提前获得航空公司 Special Cargo Acceptance
严禁：将危险品以"普通货物"名义申报（俗称"隐瞒危险品"）是重大违法行为，一经查获将面临航空公司拒载、海关处罚、货物扣押，情节严重可追究刑事责任。
💎
VAL

贵重品

▾

定义与范围

  • 单件价值超过 USD 1,000/kg 或单票总价值超过 USD 100,000 的货物
  • 包括：珠宝、黄金、铂金、钻石、名表、现金及有价证券
  • 高价值电子产品（iPhone 整批）、奢侈品、艺术品、文物

特殊处理要求

  • 须提前向航空公司申报贵重品，并获得 VAL 接受确认（VAL Acceptance）
  • 须购买货物运输险（Marine Cargo Insurance），保额覆盖申报价值
  • 在机场须走贵重品专用通道（Vault），不可普通货物装卸
  • 转机时须特别跟踪，避免在中转机场遗失
  • 文件须注明 AWB 上的 Declared Value for Carriage
AWB 申报价值（Declared Value for Carriage）直接影响航空公司的赔偿责任上限（华沙/蒙特利尔公约），建议真实申报，勿低报以节省保险费——一旦货损，低申报将导致大幅减少赔偿金额。
🥦
PER

易腐品

▾

定义与范围

  • 在常温下易腐烂变质、需全程温控运输的货物
  • 新鲜果蔬、鲜花（切花、盆栽）、活鱼苗
  • 新鲜肉类、海鲜、乳制品
  • 生物制品、疫苗、血液制品、医疗样本
  • 部分高端化妆品、营养品

操作要求

  • 须提交"易腐品声明"（Perishable Statement），注明品名、温度要求、预计运输时间
  • 包装须具备足够绝热性能，内含足量蓄冷剂（冰袋/干冰）
  • 航空公司须优先安排，缩短地面等待时间（Minimize Ground Time）
  • 药品/疫苗类须符合 GDP（Good Distribution Practice）要求，全程记录温度
  • 目的港清关须尽快提取，避免在仓库积压导致变质
干冰（Dry Ice / UN1845）属于危险品 Class 9，在用作易腐品保温介质时须同时满足 PER 和 DGR 要求：单件干冰用量 ≤ 2.5 kg 且须在 AWB 上注明干冰重量。
🐾
AVI

活体动物

▾

适用范围

  • 哺乳动物（猫、狗等宠物）、禽类、爬行类、鱼类、昆虫
  • 实验室动物、濒危动物（须提供 CITES 许可证）

必备文件

  • 兽医健康证明（Veterinary Health Certificate），出发前 10 天内开具
  • 进出口许可证（目的国动物检疫许可）
  • CITES 证书（濒危物种须提供）
  • IATA 活体动物运输规则（LAR）符合性声明

操作要求

  • 航空箱须符合 IATA 第 43 章活体动物容器标准
  • 须控制货舱温度（通常 15–25°C），并通知机组
  • 转机时须得到地面代理特殊照料，不可长时间停留在机坪暴晒
各航空公司对可运输动物种类有不同限制（如部分航司不接受宠物随机托运）。订舱前须逐一确认目标航班的 AVI 接受政策。
🏗️
OHG / HEA

超规格 / 重型货物

▾

定义标准

  • OHG（超尺寸）：单件超出标准 ULD 装载范围，需使用主甲板或特种集装器
  • HEA（重型）：单件重量超过 150 kg
  • 两者往往同时出现，需申请特种舱位（Charter 或宽体客机腹舱）

操作要点

  • 须提供货物三维尺寸（精确到厘米）和精确毛重
  • 航空公司评估后给出"舱位可行性确认"（Space & Weight Feasibility）
  • 地面操作须使用叉车或特种装卸设备，提前安排地勤
  • AWB 须注明单件重量及超规格标记
超规格货物的运费通常按"整板计费"而非按重量，且会产生额外的 OHG 附加费（Surcharge）。须在报价阶段明确告知客户，避免事后产生争议。
⚠️ 法规依据：以下分类依据 IATA 危险品规则（DGR）第 65 版（2024年）及《中国民用航空危险品运输管理规定》（CCAR-276）。货代人员须每 2 年完成一次 IATA DGR 培训并获得证书，方可处理危险品货物。
Class
1

爆炸品（Explosives）

六个次要危险分类 · 烟花/弹药/引爆器
▾

次要分类

  • 1.1 整体爆炸危险（炸药、雷管）
  • 1.2 抛射危险但无整体爆炸危险（子弹）
  • 1.3 火灾危险（烟火）
  • 1.4 无重大危险（安全烟火）
  • 1.5 不敏感的整体爆炸物
  • 1.6 极不敏感爆炸物

货代常见货物

  • 烟火产品（部分 1.3G / 1.4G）：须出口许可证
  • 安全气囊模块（内含少量炸药）：Class 1.4
  • 引爆装置、礼花炮
❌ 1.1 / 1.2 / 1.3 禁止客机运输 ⚠️ 1.4 / 特定例外 — 限货机
绝大多数 Class 1 货物禁止客机（Passenger Aircraft）运输，部分允许货机（Cargo Aircraft Only，CAO）运输。须逐一核查具体 UN 编号。
Class
2

气体（Gases）

2.1 易燃 · 2.2 非易燃非毒性 · 2.3 毒性
▾

三个子类

  • 2.1 易燃气体：丁烷、丙烷、氢气、打火机充气罐
  • 2.2 非易燃非毒性气体：液氮、氦气、CO₂（干冰气化）、压缩空气
  • 2.3 毒性气体：氯气、光气（极少商业运输）

货代常见货物

  • 灭火器（CO₂ 或干粉）：Class 2.2
  • 医疗氧气瓶：Class 2.2，须医生处方
  • 气体打火机/喷雾剂：Class 2.1 或 2.2
  • 雪地摩托、汽车用气体发生器
⚠️ 2.1 限量或 CAO · 2.2 部分允许客机 ❌ 2.3 通常禁止商业航班
Class
3

易燃液体（Flammable Liquids）

闪点 ≤ 60°C 的液体 · 最常见 DG 类别
▾

常见货物（货代最高频）

  • 工业用酒精（乙醇）：≥70% 浓度属 Class 3
  • 香水、古龙水（含酒精）：PI 970 / PI 971 限量规定
  • 油漆、涂料、清漆、稀释剂
  • 汽油、柴油、航空燃料
  • 工业溶剂（丙酮、甲苯）
  • 胶水、粘合剂（含有机溶剂）

包装要求

  • 按包装等级（PG I / II / III）选择对应包装指令（PI）
  • PG I（高危）：如汽油，须使用 UN 认证钢桶
  • PG III（低危）：如香水，可使用 PI 970 例外限量
⚠️ PG I/II 通常 CAO · PG III 限量可客机
香水等消费品通常按 PI 970（例外数量）或 PI 971 操作，单件不超过 1L 总计不超过 24 件时可走普通客机，但必须正确贴标。
Class
4

易燃固体 / 自燃 / 遇水易燃

4.1 易燃固体 · 4.2 自燃 · 4.3 遇水
▾
  • 4.1 易燃固体：硫磺、赛璐珞、活化镁粉、安全火柴
  • 4.2 自燃物质：白磷、有机金属化合物（遇空气燃烧）
  • 4.3 遇水放出易燃气体：金属钠、钾、电石（CaC₂）
  • 货代实际操作中相对少见，但一旦出现须格外谨慎
  • 4.3 货物须防止包装破损进水，存储须绝对干燥环境
⚠️ 4.1 部分限量 · 4.2 / 4.3 通常 CAO
Class
5

氧化剂 / 有机过氧化物

5.1 氧化剂 · 5.2 有机过氧化物
▾
  • 5.1 氧化剂：硝酸铵肥料、氯酸盐、过硼酸钠（洗衣粉原料）、漂白粉
  • 5.2 有机过氧化物：过氧化苯甲酰（塑料固化剂）、过氧化甲乙酮
  • 5.1 本身不燃烧，但能助燃，与可燃物接触极危险
  • 5.2 可能自加速分解，须严格控温（冷冻存储）
  • 货代常见：含氧化剂的护肤品、过氧化氢（双氧水）消毒品
⚠️ 5.1 低危类限量客机 · 5.2 通常 CAO
Class
6

毒性 / 感染性物质

6.1 毒性 · 6.2 感染性物质（生物材料）
▾
  • 6.1 毒性物质：农药、杀虫剂、有机磷化合物（UN2783 等）
  • 6.2 感染性物质：A 类（UN2814/UN2900，能致命疾病）、B 类（UN3373，诊断样本）
  • 医院诊断用血液样本通常为 UN3373（B 类），须使用 P650 三层包装
  • 新冠、流感等病毒样本属 A 类，须专业机构处理，普通货代不接受
  • 农化产品（农药）是 6.1 中货代最常遇到的货物
⚠️ 6.1 PG III 限量客机 · 6.2 A 类严格限制
Class
7

放射性物质（Radioactive）

UN2908–UN3332 · 需专业机构处理
▾
  • 医疗放射性同位素（如碘-131、锝-99m）：常见于医疗设备货物
  • 工业仪器中的放射源（如料位计、测厚仪）
  • 须具备原子能管理部门颁发的运输许可证
  • 包装须有 Class 7 标签（三叶草标志），按辐射水平分 I-White / II-Yellow / III-Yellow
  • 与其他 DG 货物须保持安全隔离距离
  • 通常须提前 72 小时向航空公司申报
⚠️ 极严格限制，普通货代几乎不处理
Class
8

腐蚀品（Corrosives）

酸类 · 碱类 · 电解质液体
▾
  • 强酸：盐酸、硫酸、硝酸（高浓度）
  • 强碱：氢氧化钠溶液、氢氧化钾
  • 湿电池（含酸电解液）：铅酸蓄电池（UN2794）
  • 电镀化学品、工业清洁剂（次氯酸钠）
  • 腐蚀品泄漏会损坏飞机蒙皮和其他货物，须严格内衬防漏包装
  • 货代常见：铅酸电池（蓄电池）运输，须排液或满足 Spec 58 标准
⚠️ PG I / II 通常 CAO · PG III 限量客机
湿电池（铅酸蓄电池）须按 PI 806 / PI 807 操作，须在包装上标注"电池含腐蚀性电解液"，且须与锂电池（Class 9）区分处理。
Class
9

杂项危险品（Miscellaneous）

锂电池 · 干冰 · 磁性物质 · 环保危害品
▾

货代最高频！锂电池（最重要）

  • 锂离子电池（UN3480）：单独运输的锂离子电池，限货机，荷电状态（SOC）≤ 30%
  • 锂离子电池（UN3481）：随设备携带的锂离子电池（手机、笔记本），可客机，限量
  • 锂金属电池（UN3090/3091）：一次性电池，规定更严，多数航司限货机
  • Wh（瓦时数）是关键参数：≤100Wh 为小电池，100–160Wh 为中电池，>160Wh 须特别许可

其他 Class 9 货物

  • 干冰（UN1845）：单件 ≤ 2.5 kg 可客机，须标注重量
  • 磁性物质：指南针偏转范围检测合规后方可装机
  • 干冰类车辆：电动汽车整车运输（含锂电）
⚠️ 锂电池规则复杂，每年 IATA DGR 均有更新，须年度复训
重点！锂电池是目前货代最常遇到且最容易违规的 Class 9 货物。电商客户寄发的消费电子、充电宝几乎全部含锂电，必须在接单时核查是否满足相关 PI。

为什么温控如此重要？

温度偏差是造成医药品失效、食品变质、生物样品失活的首要原因。WHO 数据显示，全球约 25% 的疫苗因冷链断链而失效。货代作为运输链条的组织者，须确保从出发地到目的地的全程温度符合要求。

货代的温控责任边界

货代负责选择合适的运输方案（航班时效、中转港气候、地面操作时间）和协调温控容器（EPS 箱、VIP 箱、冷链车）；客户/托运人负责提供货物温度要求文件和正确包装。双方共同对全程温控合规负责。

🏥 GDP（Good Distribution Practice）

药品经营质量管理规范，由 WHO 和各国药监局（中国：NMPA）制定，规定了药品在整个分销链中的质量保证要求。

  • 货代须建立 SOP 覆盖温控货物全程操作
  • 须对温控设备（温度记录仪、冷藏车）定期校准
  • 人员须完成 GDP 培训并留存记录
